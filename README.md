# react-redux-ReactiveX
## Installation
Please use yarn   
`yarn install`   
or   
`npm install`   
## Dev   
To launch webpack dev serveur   
`npm run dev`   
## Prod
To build   
`npm run build`   
To launch the server serving the front   
`npm run prod`
